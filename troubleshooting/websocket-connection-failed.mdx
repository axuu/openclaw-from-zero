# WebSocket 连接失败

## 问题描述

连接 Gateway 时出现 WebSocket 错误：

```bash
Error: WebSocket connection failed
```

## 解决方案

### 方案 1: 确认 Gateway 正在运行

```bash
openclaw gateway status
```

### 方案 2: 检查网络配置

确保使用正确的地址：

```bash
ws://localhost:8080
```

### 方案 3: 检查代理设置

如果你使用代理，可能需要配置排除本地地址：

```bash
export NO_PROXY=localhost,127.0.0.1
```

### 方案 4: 重启 Gateway

```bash
openclaw gateway stop
openclaw gateway start
```

### 方案 5: 检查 SSL 配置

如果使用 WSS，确保证书配置正确。
