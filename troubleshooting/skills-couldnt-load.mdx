# Skills 无法加载

## 问题描述

运行 Skill 时出现加载错误：

```bash
Error: Skills couldn't load: validation failed
```

## 解决方案

### 方案 1: 检查 Skill 文件格式

确保 Skill 文件格式正确：

```bash
openclaw skills validate
```

### 方案 2: 检查文件路径

Skills 应该放在 `.openclaw/skills/` 目录下。

### 方案 3: 检查语法错误

常见的语法问题：

- YAML 缩进错误
- 缺少必要字段
- 关键字拼写错误

### 方案 4: 清除缓存

```bash
openclaw skills clear-cache
openclaw skills reload
```
