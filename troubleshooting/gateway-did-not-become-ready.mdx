# Gateway 未就绪

## 问题描述

启动 Gateway 时卡住或超时：

```bash
openclaw gateway start
# Error: Gateway did not become ready within 30 seconds
```

## 解决方案

### 方案 1: 检查端口占用

```bash
lsof -i :8080
```

如果端口被占用，终止占用进程或更改配置中的端口。

### 方案 2: 检查防火墙

确保防火墙没有阻止本地连接：

```bash
# macOS
sudo /usr/libexec/ApplicationFirewall/socketfilterfw --add /usr/local/bin/openclaw
```

### 方案 3: 增加超时时间

在配置文件中增加超时设置：

```yaml
gateway:
  timeout: 60s
```

### 方案 4: 查看详细日志

```bash
openclaw gateway start --verbose
```
