# 润色语音日志

将用户通过语音输入的原始文字，修正识别错误并润色为可读的日志记录。

## 输入

用户直接粘贴或输入语音转文字的原始内容。如果用户没有提供内容，询问用户。

## 处理规则

1. **修正语音识别错误**：同音字、多字漏字、标点缺失
2. **保留原意**：不添加原文没有的信息，不改变表达的意思
3. **适当分段**：按话题或时间自然断开，每段之间空一行
4. **简洁为主**：
   - 不加大标题小标题
   - 不用列表、表格等复杂格式
   - 去掉口语化的填充词（嗯、那个、就是说）
   - 保持口吻自然，不要书面化过度
5. **输出格式**：纯文本，直接给出润色后的结果，不需要解释改了什么
