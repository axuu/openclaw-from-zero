# 配合 Claude Code 使用

Claude Code 是一个强大的 AI 编程助手，可以与 OpenClaw 集成。

## 安装

```bash
# 安装 Claude Code CLI
npm install -g @anthropic-ai/claude-code

# 安装 OpenClaw MCP 服务器
openclaw mcp install
```

## 配置

在 Claude Code 配置文件 `~/.claude/config.json` 中添加：

```json
{
  "mcpServers": {
    "openclaw": {
      "command": "openclaw",
      "args": ["mcp", "start"]
    }
  }
}
```

## 使用方式

启动 Claude Code 后，可以直接使用 OpenClaw 的功能：

```
> 请帮我检查今天的水质数据
```

Claude Code 会自动调用 OpenClaw 的相关 Skills。

## 可用工具

- `check_water_quality` - 检查水质
- `get_temperature` - 获取温度
- `run_skill` - 运行指定 Skill
