# 配合 Obsidian 使用

Obsidian 是一个强大的知识管理工具，与 OpenClaw 配合使用可以提升效率。

## 安装插件

1. 打开 Obsidian 设置
2. 进入社区插件
3. 搜索 "OpenClaw"
4. 安装并启用

## 配置连接

在 Obsidian 的 OpenClaw 插件设置中：

1. 填写 Gateway 地址：`ws://localhost:8080`
2. 点击 "Test Connection"
3. 保存设置

## 使用方式

### 快捷命令

- `Cmd/Ctrl + P` 打开命令面板
- 输入 "OpenClaw: Run Skill"
- 选择要执行的 Skill

### 模板集成

你可以在 Obsidian 模板中使用 OpenClaw 变量：

```markdown
---
water_quality: {{openclaw.water-quality}}
last_check: {{openclaw.last-check-time}}
---
```

## 技巧

- 使用 Daily Notes 自动记录养殖日志
- 配合 Dataview 插件分析数据
