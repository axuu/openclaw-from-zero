# 本地开发工作流

## 环境准备

### 必需软件

- Node.js 18+
- Git
- OpenClaw CLI

### 安装依赖

```bash
# 克隆仓库
git clone https://github.com/openclaw/my-project.git
cd my-project

# 安装依赖
npm install
```

## 开发流程

### 1. 启动开发服务器

```bash
# 终端 1: 启动 Gateway
openclaw gateway start

# 终端 2: 启动前端开发服务器
npm run dev
```

### 2. 开发 Skills

在 `.openclaw/skills/` 目录下创建或修改 Skills。

### 3. 测试

```bash
# 运行测试
npm test

# 运行 Skill 测试
openclaw skills test
```

### 4. 调试

```bash
# 启用详细日志
openclaw gateway start --verbose

# 查看日志
openclaw logs tail
```

## 最佳实践

1. **版本控制** - 将 Skills 纳入 Git 管理
2. **自动化测试** - 为每个 Skill 编写测试
3. **文档同步** - 保持代码和文档一致
